# Machine Learning

## About
Implementation of some fundamental machine learning models and algorithms in Python. The goal of this project is not to write the most optimized and efficient algorithms, but rather to understand how they work from the inside.

## Implementations
### Supervised Learning
- [Intro](archml/ML_Intro.ipynb) - Foundation of Machine Learning
- [Regression and Regularization](archml/ML_Regression_and_Regularization.ipynb) - Implementation of Linear Regression, Ridge, Lasso, and ElasticNet models.
- [Validation and Optuna Tuning](archml/ML_Validation_and_Optuna_Tuning.ipynb) - Implementation cross-validation strategies, hyperparameter tuning with **Optuna**, and feature importance analysis via **SHAP**.
- [Classification Basics](archml/ML_Classification_Basics.ipynb) - Implementation Logistic Regression and KNN.
- [Decision Trees and Boosting](archml/ML_Decision_Trees_and_Boosting.ipynb) - Implementation Decision Tree, Random Forest and some Boosting models

### Unsupervised Learning
- [Dimensionality Reduction and Clustering](archml/ML_Dimensionality_Reduction_and_Clustering.ipynb) - Data compression and visualization. Applying PCA, SVD, and UMAP to high-dimensional datasets.
- [Clustering Algorithms](archml/ML_Clustering_Algorithms.ipynb) - Implementation KMeans, DBSCAN and applying Agglomerative Clustering and Gaussian Mixture Models.
